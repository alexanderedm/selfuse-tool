# 基本使用

本頁面介紹 Selftool 的基本操作與日常使用方式。

---

## 🎯 首次設定

### 1. 設定音訊裝置

首次啟動 Selftool 後，需要先設定要切換的兩個音訊裝置：

1. **開啟設定視窗**

    右鍵點擊系統托盤圖示 → 選擇「⚙️ 設定」

2. **選擇裝置**

    - **裝置 A**：選擇你的第一個音訊裝置（例如：喇叭）
    - **裝置 B**：選擇你的第二個音訊裝置（例如：耳機）

3. **自動儲存**

    設定會即時儲存，關閉視窗即可

!!! tip "提示"
    如果看不到你的裝置，請確認：

    - 裝置是否已正確連接
    - Windows 系統是否已識別該裝置
    - 嘗試重新啟動 Selftool

---

## 🔄 音訊裝置切換

### 快速切換

設定完成後，切換音訊裝置非常簡單：

1. 右鍵點擊托盤圖示
2. 選擇「🔄 切換輸出裝置」
3. 系統會自動切換並顯示通知

### 托盤圖示顏色

圖示顏色會隨著當前裝置改變：

| 顏色 | 裝置狀態 |
|------|----------|
| 🔵 藍色 | 目前使用裝置 A |
| 🟢 綠色 | 目前使用裝置 B |
| ⚫ 灰色 | 使用其他裝置或未設定 |

## 🎵 音樂播放器

### 開啟音樂播放器

右鍵托盤圖示 → 選擇「🎵 本地音樂播放器」

### 基本操作

1. **新增音樂**

    點擊「選擇資料夾」→ 選擇包含音樂檔案的資料夾

2. **播放控制**

    - ⏯️ 播放/暫停
    - ⏭️ 下一首
    - ⏮️ 上一首
    - 🔀 隨機播放
    - 🔁 循環播放

3. **歌詞顯示**

    在音樂檔案同目錄放置 `.lrc` 檔案（與音樂檔案同名），即可自動顯示同步歌詞

4. **元數據補全**

    播放時自動從 YouTube Music 和 iTunes 抓取專輯封面、藝術家、專輯名稱

### 進階功能

- **搜尋與過濾**：使用搜尋框快速找到想聽的歌曲
- **播放清單**：建立自訂播放清單
- **歷史記錄**：查看最近播放的歌曲
- **YouTube 下載**：直接從 YouTube 下載音樂
- **Discord Rich Presence**：在 Discord 顯示正在播放的歌曲

## 📰 RSS 閱讀器

### 訂閱 RSS

1. 開啟 RSS 閱讀器：右鍵托盤圖示 → 「📰 RSS 閱讀器」

2. 點擊「新增訂閱」

3. 輸入 RSS URL（例如：`https://example.com/feed`）

4. 點擊「訂閱」

### 管理訂閱

- **刷新**：點擊「刷新」更新所有訂閱的最新文章
- **取消訂閱**：選擇訂閱源 → 點擊「刪除」
- **閱讀文章**：點擊文章標題在右側預覽，或在瀏覽器中開啟

## 🤖 AI 瀏覽器助手

!!! warning "需要 OpenAI API Key"
    使用 AI 瀏覽器助手前，需要設定 `OPENAI_API_KEY` 環境變數

### 設定 API Key

**方法一：系統環境變數**

1. 開啟「系統內容」→「進階」→「環境變數」
2. 新增變數：
   - 變數名稱：`OPENAI_API_KEY`
   - 變數值：`sk-your-api-key-here`
3. 重新啟動 Selftool

**方法二：透過配置文件**（即將推出）

支援在設定視窗中直接輸入 API Key，使用 Windows DPAPI 加密儲存。

### 使用助手

1. 右鍵托盤圖示 → 「🤖 AI 瀏覽器助手」
2. 在 UI 中輸入指令，AI 會自動操作瀏覽器

## 📊 查看統計

想知道你的音訊裝置使用習慣？

1. 右鍵托盤圖示 → 「📊 使用統計」

2. 查看：
   - 每個裝置的使用時間
   - 切換次數統計

## ⚙️ 其他設定

### 音樂播放器設定

- **音量調整**：使用音量滑桿調整播放音量
- **元數據補全開關**：設定 → 啟用/停用自動補全功能

### RSS 設定

- **快取時間**：控制 RSS 刷新頻率
- **文章數量限制**：限制每個訂閱源顯示的文章數量

## 💡 使用技巧

### 技巧 1：快速切換音訊

如果經常切換裝置，可以：

1. 將 Selftool 設為開機啟動
2. 記住快捷鍵（未來支援）
3. 將托盤圖示固定顯示（不要隱藏到溢位區）

### 技巧 2：音樂庫管理

為了獲得最佳體驗：

1. 將音樂檔案命名清楚（藝術家 - 歌曲名稱.mp3）
2. 準備 `.lrc` 歌詞檔案以啟用同步歌詞
3. 啟用元數據自動補全功能

### 技巧 3：RSS 高效閱讀

- 使用搜尋功能快速找到感興趣的文章
- 定期清理舊訂閱源
- 善用文章過濾功能
